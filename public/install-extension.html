Installation Guide

1

Download the Extension

Click the download button above. A file named kiara-vision-extension.zip will be saved to your Downloads folder.

2

Open Chrome Extensions

Open Chrome (or Edge/Brave) and type chrome://extensions in the address bar. Press Enter.

3

Enable Developer Mode

Look for the "Developer mode" toggle in the top-right corner and turn it ON.

4

Drag & Drop the ZIP File

Drag the kiara-vision-extension.zip file from your Downloads folder directly onto the extensions page. Chrome will automatically install it!
